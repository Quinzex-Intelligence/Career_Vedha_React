# Career Vedha Frontend

A premium, modern, and highly interactive Career Guidance and Portal management system built with React. This platform provides a comprehensive suite of tools for students to explore careers, take exams, and for administrators to manage users, roles, and notifications.

## 🚀 Key Features

- **Dashboard & Analytics**: Real-time overview of portal activity with premium glassmorphism design.
- **Career Exploration**: Discover various career paths and educational opportunities.
- **Exam & Quiz Module**: Robust system for taking and managing academic and competitive exams.
- **User Management**: Comprehensive admin tools for managing active/inactive users and role approvals.
- **Role-Based Access Control (RBAC)**: Fine-grained permissions and role management for Super Admins and Admins.
- **Real-time Notifications**: Instant alerts for system actions and user requests with processed/archive tracking.
- **Global Search**: Deep-link search across the entire portal for quick navigation.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS (Premium Custom Design System)
- **State Management**: React Context API & Hooks
- **Communication**: Axios (REST API), WebSockets (Real-time updates)
- **Visuals**: Lucide Icons, FontAwesome, Google Fonts (Inter)

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/annemchaitanyakumar/Career_Vedha_Frontend.git

# Navigate to project directory
cd Career_Vedha_Frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_BASE_URL=http://your-backend-api-url
```
